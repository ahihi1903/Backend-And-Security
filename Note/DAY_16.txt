3️⃣ JWT là gì?
	JWT(JsonWebToken):header.payload.signature(tiêu đề.tải trọng.chữ ký)
	vd:eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
	vì có chữ ký signature nên ko thể fake

JWT (JSON Web Token) là một chuỗi string dùng để:
✔️ xác thực user (đã login chưa)
✔️ truyền thông tin an toàn giữa client ↔ server

🔟 Flow mới (QUAN TRỌNG)
REGISTER
password → hash → lưu DB

LOGIN
password → compare hash
 ↓
tạo JWT

REQUEST
client gửi token
 ↓
server verify signature
 ↓
OK → cho vào controller
