DAY 5 – Middleware System

Middleware là function nằm giữa request và controller.
----------
Flow hiện tại của bạn:
request
   ↓
matchRoute
   ↓
controller
   ↓
response
---------
Sau khi thêm middleware:
request
   ↓
matchRoute
   ↓
middleware 1
   ↓
middleware 2
   ↓
controller
   ↓
response
<------------------>
Middleware có dạng:

function middleware(req, res, next) {
  console.log("middleware chạy");

  next(); // chuyển sang bước tiếp
}

next() = chạy function tiếp theo.

<------------>
function.length trong JS = số tham số của function.
<------------->
