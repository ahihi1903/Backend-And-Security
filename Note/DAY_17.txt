DAY-17:Access Token + Refresh Token

1️⃣ Vấn đề của Day 16

Bạn đang có:

JWT expiresIn = 1h

Sau 1 giờ:

token hết hạn
user phải login lại

❌ Trải nghiệm không tốt.

2️⃣ Giải pháp thực tế công ty dùng

Dùng 2 token:

1. Access Token  (ngắn hạn)
2. Refresh Token (dài hạn)


3️⃣ Hiểu cực dễ
Token		Sống bao lâu	Dùng để
Access Token	15 phút		gọi API
Refresh Token	7 ngày		xin access token mới

4️⃣ Flow chuẩn
LOGIN
 ↓
server trả:
- accessToken
- refreshToken

Client gọi API:
Authorization: Bearer accessToken

Hết hạn:
POST /refresh

Server cấp accessToken mới

11️⃣ Vì sao mạnh?
Access token bị lộ → chỉ sống ngắn
Refresh token giữ đăng nhập lâu

12️⃣ Sai lầm newbie
❌ token sống 30 ngày
❌ không refresh token
❌ hết hạn bắt login lại
