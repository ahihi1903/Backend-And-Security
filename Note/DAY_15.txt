DAY-15:Authentication (đăng nhập + token + bảo vệ API)
✔ Login system
✔ Token (giống JWT)
✔ Middleware bảo vệ route
✔ Hiểu cách auth hoạt động thật

-Tư duy: Client login → Server tạo token → Client gửi token mỗi request

-Flow:
POST /login
 ↓
server check user
 ↓
tạo token
 ↓
trả về client
 ↓
client gọi API khác kèm token
 ↓
middleware check token
 ↓
cho vào controller
